equalHeights
============

equal height items

name:      equalHeights
purpose:   to maintain equal heights across columns 
usage:     markup required is:
`<div class="equal-height-group">
  <div class="equal-height-item"></div>
  <div class="equal-height-item"></div>
</div>`

options:   el:   a jQuery element object
           opts: `{cutoff: integer}`


requires: jQuery and underscore
